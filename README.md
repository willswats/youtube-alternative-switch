# YouTube Alternative Switch

A Firefox extension that adds context menu buttons for switching between YouTube and Piped, as well as YouTube and Invidious.

## Packaging

CD to `src` and run:

```bash
zip -r -FS ../youtube-alternative-switch.xpi *
```
